# Blinkit Sales Dashboard | Power BI Project

## Project Overview

The Blinkit Sales Dashboard is a business intelligence project developed using Power BI to analyze sales performance, customer behavior, and product trends. The goal of this project is to transform raw sales data into meaningful insights that help understand business performance and support decision-making.

The dashboard provides a clear overview of sales distribution, product performance, outlet performance, and customer ratings.

## Objectives

The main objectives of this project are:

* Analyze total sales and revenue performance.
* Identify top-performing product categories.
* Understand outlet performance across different locations.
* Analyze customer rating patterns.
* Track sales trends over time.

## Tools and Technologies Used

* Power BI
* Data Cleaning and Transformation
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Key Dashboard Features

### Total Sales and Revenue Analysis

Provides an overview of overall business performance including total revenue, number of orders, and quantity sold.

### Category-wise Sales Performance

Analyzes which product categories contribute the most to total revenue.

### Outlet Type and Location Analysis

Examines sales performance across different outlet types and geographic locations.

### Customer Rating Insights

Analyzes customer satisfaction through rating distribution.

### Top Performing Products

Identifies the best-selling products based on sales and quantity.

### Sales Trend Analysis

Tracks sales patterns over time using a time-series visualization.

## Dashboard Pages

### Page 1 – Sales Overview

Displays key performance indicators including total sales, total orders, average ratings, and quantity sold.

### Page 2 – Category Performance

Analyzes category-wise and product-level performance.

### Page 3 – Outlet Analysis

Shows sales distribution across outlet types and locations.

### Page 4 – Customer Insights

Visualizes customer ratings and product popularity.

### Page 5 – Sales Trends

Displays sales growth and monthly trends.

## Insights Gained

* Certain product categories contribute significantly to overall revenue.
* Outlet performance varies based on location and outlet type.
* Customer ratings provide valuable insights into product quality.
* Sales trends reveal seasonal patterns in purchasing behavior.

## Conclusion

This dashboard demonstrates how Power BI can be used to transform raw data into interactive visualizations and actionable business insights. The project highlights key skills in data analysis, dashboard design, and business intelligence.
